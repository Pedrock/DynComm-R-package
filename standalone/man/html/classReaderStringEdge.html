Detailed Description

Reader for edges from string.

Format is one edge per line, with values separated by any white space (space or tab) in any amount. This Reader can not read backwards. Example valid string: "1 2\n#comment\n\n1 3\n2 3\n3 6\n4 6\n4 5\n5 7\n6 7"
